240 /*
241 * Register a periodic callback for this bus. All callbacks on the
242 * same bus are made from the same thread with lock already taken. In
243 * other words, the callback is not executed on the main thread (or the
244 * thread which registered the callback), but in a separate per-bus
245 * thread.
246 *
247 * After registering the periodic callback, the other functions should not
248 * be used anymore from other contexts. If it really needs to be done, the
249 * lock must be taken.
250 *
251 * Return: A handle for this periodic callback. To cancel the callback
252 * call #unregister_callback() or return false on the callback.
253 */
254 virtual PeriodicHandle register_periodic_callback(uint32_t period_usec, PeriodicCb) = 0;

256 /*
