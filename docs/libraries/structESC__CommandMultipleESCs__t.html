Detailed Description

This packet can be used to simultaneously command multiple ESCs which have sequential CAN ID values. This packet must be sent as a broadcast packet (address = 0xFF) such that all ESCs can receive it. These commands can be sent to groups of ESCs with ID values up to 64, using different ESC_SETPOINT_x packet ID values.
