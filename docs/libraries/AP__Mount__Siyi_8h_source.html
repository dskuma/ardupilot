1/*
2 Siyi gimbal driver using custom serial protocol
3
4 Packet format (courtesy of Siyi's SDK document)
5
6 -------------------------------------------------------------------------------------------
7 Field Index Bytes Description
8 -------------------------------------------------------------------------------------------
9 STX 0 2 0x5566: starting mark
10 CTRL 2 1 bit 0: need_ack. set if the current data packet needs ack
11 bit 1: ack_pack. set if the current data packate IS an ack
12 bit 2-7: reserved
13 Data_len 3 2 Data field byte length. Low byte in the front
14 SEQ 5 2 Frame sequence (0 ~ 65535). Low byte in the front. May be used to detect packet loss
15 CMD_ID 7 1 Command ID
16 DATA 8 Data_len Data
17 CRC16 2 CRC16 check the complete data package. Low byte in the front
18 */
