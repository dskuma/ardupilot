Decode a ESC_WarningBits_t from a byte array.

The warning bits enumerate various system warnings/errors of which the user (or user software) should be made aware. These warning bits are transmitted in the telemetry packets such that user software is aware of any these warning conditions and can poll the ESC for particular packets if any further information is needed.
