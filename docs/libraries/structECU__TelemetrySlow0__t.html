Detailed Description

This is the first of three slower telemetry packets which are transmitted by the ECU at a user customizable period (between 0.5s and 10.0s). These three packets contain data that is not likely to change as quickly as the data in the fast telemetry packet. By default, the slow telemetry messages are transmitted at 1Hz (period of 1.0s).
