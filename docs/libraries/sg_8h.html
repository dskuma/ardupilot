Author

Date

Sagetech protocol for host message building and parsing.

This module performs both the following:

  1. Parses raw Sagetech host messages defined in the SDIM and returns a populated struct dataset of the message type.
  2. Receives a populated struct dataset of the desired host message and returns the corresponding raw message data buffer.
