30/*
31 a method to make semaphores less error prone. The WITH_SEMAPHORE()
32 macro will block forever for a semaphore, and will automatically
33 release the semaphore when it goes out of scope
34
35 Note that we have two types of semaphores. A normal semaphore can
36 only be taken once. A recursive semaphore allows for the thread
37 holding the semaphore to take it again. It must be released the same
38 number of times it is taken.
39
40 The WITH_SEMAPHORE() macro can be used with either type of semaphore
41 */
42
