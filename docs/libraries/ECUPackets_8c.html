Create the ECU_UserData packet.

Create the ECU_UserData packet from parameters.

The user data packet provides the user with 8 (eight) bytes of data for storing custom parameters or settings in ECU EEPROM (non-volatile) memory. The ECU does not make use of these values; they are simply for storing user data. Send a zero length packet to request the current settings. To set these values send the system command SET_USER_DATA. Data values must be set individually.
