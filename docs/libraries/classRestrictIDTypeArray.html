Detailed Description

template<class base_type, uint32_t num_instances, typename accessor_type>
class RestrictIDTypeArray< base_type, num_instances, accessor_type >

This template associates the base_type array with accessor_type(index). So the elements can be accessed using [] only using accessor_type index _priv_instance is kept public for use in Parameter declaration.
