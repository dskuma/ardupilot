Decode the Servo_MultiPositionCommand packet.

Decode the Servo_MultiPositionCommand packet to parameters.

This packet can be used to simultaneously command multiple servos which have sequential CAN ID values. This packet must be sent as a broadcast packet (address = 0xFF) such that all servos can receive it. These commands can be sent to groups of servos with ID values up to 64, using different PKT_SERVO_MULTI_COMMAND_x packet ID values.

Parameters
_pg_pktpoints to the packet being decoded by this function
commandAreceives Servo command for servo with address offset 0
commandBreceives Servo command for servo with address offset 1
commandCreceives Servo command for servo with address offset 3
commandDreceives Servo command for servo with address offset 3
Returns
0 is returned if the packet ID or size is wrong, else 1
