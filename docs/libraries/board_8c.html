void __early_init ( void )

STM32F100_MCUCONF.

Early initialization code.

This initialization must be performed just after stack setup and before any other initialization.

You must not rely on: 1) BSS variables being cleared 2) DATA Variables being initialized 3) RAM functions to be in RAM You can rely on: 1) const variables or tables 2) flash code 3) automatic variables
